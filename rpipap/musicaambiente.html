Vamos começar a Programar







2. Para começar, vamos verificar o circuito usando um programa simples.
     No terminal, crie um novo script: 


nano test.py

2.1. Depois de aberto, adicione este código ao arquivo e pressione Ctrl-C para sair, seguindo a caixa de diálogo Salvar: 


import RPi.GPIO  as GPIO
import time
GPIO.setwarnings(False)
GPIO.setmode(GPIO.BOARD)
GPIO.setup(7,GPIO.OUT)
GPIO.setup(11,GPIO.IN)
GPIO.setup(13,GPIO.IN)
GPIO.setup(15,GPIO.IN)

While True:
        GPIO.output(7,GPIO.HIGH)
        if (GPIO.input(11)==1):
                print("Button on GPIO 11")
                time.sleep(1)

        if (GPIO.input(13)==1:
                print("Button on GPIO 13")
                time.sleep(1)

        if (GPIO.input(15)==1):
                print("Button on GPIO 15")
                time.sleep(1)

GPIO.cleanup( )

Este código lê o valor dos botões e imprime na tela quando eles mudam. Os dois botões de pressão possuem resistores de pressão, e relatam ALTO quando pressionados. O interruptor do íman magnético informa ALTO sempre que o íman que o acompanha estiver próximo.

2.2. Execute o programa:


Python test.py

Deve ver o LED acender e um relatório na tela quando pressionar os dois botões e puxa o íman para longe do interruptor magnético. Se não estiver a receber nenhum feedback ao pressionar os botões, verifique o circuito novamente e verifique se tudo está conectado como devia.
Quando terminar o teste, pressione CTRL-C para interromper o programa.

O argumento inicial = GPIO.LOW na configuração do pino 7 define-o como baixo assim que é inicializado.

2.8. De volta ao nosso código, precisamos de iniciar o misturador Pygame e carregar a música.


pygame.mixer.init(44100,-16,2,1024)
pygame.mixer.music.set_volume(1.0)
name = "example.mp3"
pygame.mixer.music.load(name)
print("Loaded track - "+ str(name))

A primeira linha deste bloco inicia o misturador Pygame . O último argumento entre parênteses é o buffer que controla a latência na qual o som é reproduzido.

Se ao fazer o download do arquivo de música e o colocou num diretório diferente do programa, precisará fornecer o caminho completo dentro dos parênteses ao definir a variável name.
